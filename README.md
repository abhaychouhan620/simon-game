# Simon Game 🎮

A classic Simon Says memory game built using **HTML**, **CSS**, and **JavaScript**. Test your memory by repeating the sequence of colors. The sequence gets longer with each level, making the game more challenging.

---

## ✨ Features

- 🎲 Random color sequence generation
- 📈 Level-by-level progression
- ✅ User input validation
- ✨ Flash animation for button clicks
- ❌ Game over detection
- 🔄 Restart the game by pressing any key

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🎮 How to Play

1. Press any key to start the game.
2. Watch the button that flashes.
3. Repeat the sequence by clicking the colored buttons.
4. Each level adds one new color to the sequence.
5. The game ends if you click the wrong button.
6. Press any key to start a new game.
